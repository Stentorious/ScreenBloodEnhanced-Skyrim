# Screen Blood Enhanced
<p align="left"> <img height="350px" src="https://staticdelivery.nexusmods.com/mods/1704/images/175120/175120-1773989934-69224937.png"> </p>

## Overview
- Complete overhaul of the game's screen blood system.
- Allows you to customize when and how screen blood is applied.
- Easily customize the effect appearance or disable it entirely.
- Adds Metro Exodus inspired screen wiping allowing you to clear screen blood at any time.

### Screen Wiping
- Unique first person animation removing all active screen blood.
- Keyboard: H

### Enhanced Effects
- Screen blood will now apply when landing attacks on enemies at close range.
- Splatter strength is based on a large set of parameters, see 'Configuration' for details.
- Easily customize the appearance and duration of the effect.

## Configuration
Configurable via the MCM:

### Visibility
Customize screen blood visibility:
- Damage Taken: Should screen blood be applied when receiving damage.
- Damage Dealt: Should screen blood be applied when attacking enemies.

### Strength
Customize screen blood strength:
- Splatter Count: Default amount of screen blood applied before any additional multiplier.
- Weapon Type: Multipliers for Unarmed, Blunt, Blade, and Projectile weapons.
- Attack Type: Multipliers for light and heavy attacks.
- Distance: Distance from target where splatters will start to appear.
- Heading: Angle between the player and target where splatters will start to appear.

### Appearance
Customize screen blood appearance:
- Duration: Amount of time before screen blood fades away naturally.
- Blood Size: Size of the blood droplet effect.
- Blood Opacity: Opacity of the blood droplet effect.
- Flare Size: Size of the light flare effect.
- Flare Opacity: Opacity of the light flare effect.
